# Security policy

## Supported versions

Security fixes are applied to the latest released version of each component
module. Older versions are not backported.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report them privately via GitHub's
[Security Advisories](https://github.com/sunkek/samsara-components/security/advisories/new)
feature, or by emailing the maintainer directly.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected component and version

You will receive an acknowledgement within 72 hours and a resolution timeline
once the issue is confirmed.
